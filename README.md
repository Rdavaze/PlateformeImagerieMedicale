Ce projet est une plateforme de questions / réponses pour l'annotation d'imagerie médicale.

La structure est basée sur la platefome Question2Answer et ajoute la possibilité d'annoter des images médicales pour que les réponses soient plus explicites et ainsi aider la personne en question à établir un diagnostic.


Github de la plateforme Question2Answer : https://github.com/q2a/question2answer
